# vue-admin-laravel

技术栈

前端 vue-cli3 
用vuex 做的权限
laravel5 做的后端接口

前后端分离用json web token 做的安全验证


代码下载下来就可以用 

前端vue  npm install 安装依赖 

后端php 服务器 Nginx指向public/index.php 或者 php artisan serve   然后composer install 安装依赖

数据库sql 在根补录 wind.sql 直接执行 然后更改 laravel 下的数据库配置

然后 npm run dev 就可以了
